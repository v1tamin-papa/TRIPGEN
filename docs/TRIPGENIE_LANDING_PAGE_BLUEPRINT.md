## COPY GUIDELINES

### Tone & Voice
- **Primary tone**: Confident, friendly, slightly playful
- **Avoid**: Corporate jargon, overly formal language, hyperbole
- **Embrace**: Conversational contractions ("you're", "we'll"), direct address ("you", "your"), relatable frustrations
- **Humor**: Light touches okay (e.g., "Stupidly simple"), but don't force it

### Reading Level
- Target: 8th grade reading level (Flesch-Kincaid)
- Short sentences: Average 12-15 words
- Short paragraphs: 2-3 sentences max
- Active voice always

### Keywords (SEO)
Integrate naturally throughout:
- AI travel planner
- AI trip planner
- Personalized travel itinerary
- Travel planning app
- AI travel assistant
- Trip planning tool
- Automated trip planner
- Personal travel agent

### Scannability Rules
- Every section has a clear headline
- Bullet points over paragraphs where possible
- Bold key phrases within paragraphs
- White space is your friend
- Icon + text pairings for features
- Numbers and stats stand out visually

## TECHNICAL NOTES

### Mobile-First Requirements
- All sections must be tested at 375px width first
- CTAs must be thumb-reachable (bottom 60% of screen)
- Horizontal scrolling = failure
- Touch targets: Minimum 44x44px
- Hero section should fit above fold on iPhone 14 (no scroll to see CTA)

### Performance Requirements
- Largest Contentful Paint: < 2.5 seconds
- First Input Delay: < 100ms
- Cumulative Layout Shift: < 0.1
- Hero image: WebP format, max 200KB
- Lazy load everything below fold
- Critical CSS inlined

### A/B Testing Priorities
Test these elements first (highest impact):
1. **Hero headline variations**:
   - A: "Stop Planning Trips. Start Taking Them."
   - B: "Your AI Travel Agent. Ready When You Are."
   - C: "Plan Any Trip in 2 Minutes. Seriously."

2. **CTA button text**:
   - A: "Plan Your First Trip Free"
   - B: "Start Your Free Trial"
   - C: "Try TripGenie Free"

3. **Price anchoring**:
   - A: "$29/month" (simple)
   - B: "$29/month ‚Äî less than $1/day"
   - C: "$29/month ‚Äî skip one lunch, save 4 hours"

4. **Social proof format**:
   - A: Customer count ("47,000+ trips planned")
   - B: Time saved ("4.2 hours saved per trip")
   - C: Star rating + quote

### Tracking & Analytics
Implement events for:
- Scroll depth (25%, 50%, 75%, 100%)
- CTA clicks (by location: hero, sticky, final)
- Video plays (if applicable)
- FAQ accordion opens
- Pricing toggle (monthly/annual)
- Time on page
- Exit intent

### Accessibility
- All images have alt text
- Color contrast ratio: 4.5:1 minimum
- Keyboard navigation supported
- Screen reader optimized
- Focus states visible
- No content in images only
